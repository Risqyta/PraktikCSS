Font style

css dapat mengatur style dari teks yg ada pd halaman web yang dibuat. terdepat beberapa property yang dapat digunakan yaitu sebagai berikut.

  • text-align: digunakan untuk mengatur perataan teks,value yang dapat digunakan yaitu left,center,right.
  • color: digunakan untuk mengatur warna.
  • letter-spacing: mengatur jarak karakter.
  • word-spacing: mengatur jarak antar kata.
